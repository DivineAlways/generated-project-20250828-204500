Wireframe Notes

Header: Name on left, nav on right. Should be clean and simple.

Hero: High impact. Main value proposition is front and center.

About: Standard bio section. A single block of text is sufficient.

Projects: A grid layout is best for scannability. Each card needs a title, brief description, and a clear link.

Resume: Simple Call-To-Action. A button is more prominent than a simple link.

Contact: Centralized contact info. Links should be distinct.

Layout: A single-column layout for mobile will be required. The grid can stack vertically.
